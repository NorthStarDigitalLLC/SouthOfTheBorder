Full Menu Concept

Cleaner categories, better spacing, and a system built for daily updates.

The accessible public menu route shows categories for Breakfast, Lunch, Crawfish, Salad Bar, Drinks, and Dessert. This build turns that into a stronger menu experience and includes a data file you can edit.

Important

Why this is not 1:1 on the menu items yet

The live public menu loads dynamically, and the exact item-level dataset was not fully exposed in the accessible browser output. This build includes an editable JSON file so the exact items can be dropped in quickly.

How to update

Edit one file

Open data/menu.json and replace the placeholder items with the exact menu. The page will render automatically.
